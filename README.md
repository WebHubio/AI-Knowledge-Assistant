
# AI Knowledge Assistant

## Overview

AI Knowledge Assistant is a proof-of-concept web application designed to demonstrate how open-source AI can help employees quickly access internal company knowledge. The solution enables users to search company documents, SOPs, technical manuals, and internal guides using natural language.

The proposed solution uses the **Qwen** AI model to understand company documentation and provide relevant answers to employee queries.

**Note:** This project is a prototype developed as part of the **Elchai Group AI Research Intern Assessment**. It demonstrates the user interface and workflow using sample AI-generated responses.

---

## Features

* Modern responsive landing page
* AI-powered knowledge assistant concept
* Internal document search workflow
* Prototype demonstration
* Department-wise business use case
* Human-in-the-loop verification concept
