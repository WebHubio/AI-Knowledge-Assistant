## Overview

AI Knowledge Assistant is a proof-of-concept web application developed as part of the **Elchai Group AI Research Intern Assessment**.

The project demonstrates how an open-source AI model can help employees quickly search company documents, SOPs, technical manuals, and internal knowledge using natural language.

The proposed solution uses the **Qwen** AI model to retrieve relevant information from internal company documents and provide instant responses to employee queries.

**Note:** This prototype demonstrates the user interface and workflow using sample AI-generated responses. In a production environment, the Qwen model would be integrated through an API to provide real-time answers.
